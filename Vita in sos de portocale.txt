Ingrediente:

-1 borcan Amestec de legume Chop Suey 330g
-2 linguri de ulei wok
-3 linguri de sos soia
-1 catel de usturoi
-2 portocale
-200g file de vita
-ceapa verde



Se curata usturoiul si se toaca marunt. Se preseaza o portocala si se face o marinada din suc de portocale, usturoi si sos soia. Carnea de vita se taie in fasii subtiri si se lasa 15 minute in marinada.
Ceapa verde se taie in bucati lungi de 2 cm. Amestecul de legume Chop Suey se clateste intr-o sita cu apa.
Se pune uleiul wok in tigaie si se incalzeste. Se adauga carnea si se prajeste timp de 5-7 minute. Apoi, se adauga ceapa verde si amestecul de legume si se prajeste totul la foc mare. Se adauga restul de marinada si, daca este necesar, putina apa. Se mai prajeste inca 5 minute, pana cand carnea este bine patrunsa.
A doua portocala se taie rondele. Vita se decoreaza cu rondelele de portocale si se serveste cu orez basmati sau paste cu ou.
